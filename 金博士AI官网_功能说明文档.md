### 2.2 作品展示 (Portfolio)

**位置**：首页第一个区块，全屏高度

**功能说明**：

| 功能 | 描述 | 实现方式 |
|------|------|----------|
| 作品轮播 | 自动切换展示不同学员作品 | 每5秒自动切换 |
| 手动控制 | 支持左右箭头手动切换 | 点击箭头切换 |
| 指示器 | 底部显示当前作品位置 | 点击可跳转指定作品 |
| 视频播放 | 支持点击播放视频作品 | 弹出全屏视频播放器 |
| 分类标签 | 显示作品所属分类 | 动态获取分类信息 |

**数据来源**：Supabase portfolios 表

**数据字段**：
```typescript
interface Portfolio {
  id: number
  category_name: string      // 分类名称
  category_icon: string      // 分类图标
  category_color: string     // 分类颜色
  title: string              // 作品标题
  description: string        // 作品描述
  author_name: string        // 作者姓名
  author_avatar: string      // 作者头像（首字母）
  author_title: string       // 作者身份
  tags: string               // 标签（逗号分隔）
  media_type: string         // 媒体类型：video/image
  media_url: string          // 媒体地址
  media_cover: string | null // 媒体封面
  sort_order: number         // 排序序号
}
```

**交互流程**：
```
用户进入页面
    ↓
自动加载作品数据（从Supabase API）
    ↓
显示第一个作品（3秒淡入动画）
    ↓
每5秒自动轮播至下一个作品
    ↓
用户可手动切换（点击箭头或指示器）
    ↓
如为视频作品，点击播放按钮弹出视频播放器
    ↓
视频播放完毕后可关闭播放器，继续浏览
```

## 3. 用户登录功能

### 3.1 登录入口

**位置**：页脚底部"后台管理"按钮

**触发方式**：点击"后台管理"按钮

**登录弹窗**：

```
┌────────────────────────────────────┐
│           后台登录                   │
│  ┌────────────┬────────────┐       │
│  │  老师登录  │   学员登录   │       │
│  └────────────┴────────────┘       │
│                                    │
│  用户名：_________________          │
│                                    │
│  （学员登录额外显示）                │
│  所属班级：[请选择班级 ▼]            │
│                                    │
│  [登录]                            │
└────────────────────────────────────┘
```

### 3.2 老师登录流程

```
1. 用户点击"后台管理"
2. 弹窗显示登录表单（默认老师登录标签）
3. 用户输入老师姓名
4. 点击"登录"按钮
5. 系统调用 /api/login 接口
6. 后端查询飞书多维表格师资管理表
7. 匹配成功返回用户信息，跳转至 /dashboard
8. 匹配失败显示错误提示"老师姓名不存在"
```

### 3.3 学员登录流程

```
1. 用户点击"后台管理"
2. 切换至"学员登录"标签
3. 用户输入学员姓名
4. 用户选择所属班级（下拉框，数据从 /api/classes 获取）
5. 点击"登录"按钮
6. 系统调用 /api/login 接口
7. 后端查询飞书多维表格学员档案表，验证姓名+班级匹配
8. 匹配成功返回用户信息，跳转至 /dashboard
9. 匹配失败显示错误提示"学员姓名或班级信息不正确"
```

### 3.4 登录后跳转

| 用户类型 | 跳转URL | 参数 |
|----------|---------|------|
| 老师 | `/dashboard?name=XXX&type=teacher` | name, type |
| 学员 | `/dashboard?name=XXX&type=student&classId=XXX` | name, type, classId |

## 14. 错误处理

### 14.1 常见错误

| 错误场景 | 错误提示 | 处理方式 |
|----------|----------|----------|
| 登录失败-老师不存在 | "老师姓名不存在，请检查输入" | 提示用户确认姓名 |
| 登录失败-学员信息错误 | "学员姓名或班级信息不正确，请检查输入" | 提示用户确认姓名和班级 |
| 数据加载失败 | "网络错误，请稍后重试" | 显示重试按钮 |
| 操作失败 | 显示具体错误信息 | 保留表单数据 |

### 14.2 加载状态

| 状态 | 显示内容 |
|------|----------|
| 加载中 | 旋转加载图标 + "加载中..."文字 |
| 空状态 | "暂无数据" + 添加按钮 |
| 错误状态 | 错误提示 + 重试按钮 |
