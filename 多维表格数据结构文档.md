## 🔧 API 使用注意事项

### 1. 数字字段类型转换

飞书 API 要求数字字段必须传递整数类型，不能是字符串。

```javascript
// ❌ 错误：字符串格式会报 NumberFieldConvFail
fields['可带班级'] = "3"

// ✅ 正确：整数格式
fields['可带班级'] = parseInt("3", 10) || 0
```

### 2. 关联字段格式

关联字段必须传递 record_id 的字符串数组。

```javascript
// ❌ 错误：对象格式会报 LinkFieldConvFail
fields['上课班级ID'] = [{ record_ids: ["xxx"], ... }]

// ✅ 正确：字符串数组格式
fields['上课班级ID'] = ["record_id_1", "record_id_2"]
```

### 3. 日期字段格式

日期字段需要传递时间戳（毫秒）。

```javascript
// 转换日期为时间戳
const timestamp = new Date('2026-06-01').getTime()
fields['报名日期'] = timestamp
```
