## 四、数据流验证

### 4.1 用户提问数据流

```
输入: "课程有哪些"
    │
    ▼
┌─────────────────────────────────────────┐
│  验证点1: 请求体格式                      │
│  { message, sessionId?, history? }          │
└────────────────────────────────────┬───┘
                                    │
                                    ▼
                         ┌───────────────────────┐
                         │  验证点2: 消息发送  │
                         │  POST /api/chat     │
                         └───────────────────────┘
                                    │
                                    ▼
┌────────────────────────────────────────────────────────┐
│  验证点3: RAG检索                                    │
│  • getEmbedding("课程有哪些") → 生成向量              │
│  • vectorDB.search("chatbot_knowledge", vector, 3)   │
│  • 返回TOP3相关知识                                 │
└────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌────────────────────────────────────────────────────┐
│  验证点4: MiniMax对话生成                       │
│  • system prompt + 检索结果 → 构建完整prompt      │
│  • chatCompletion() → AI回答                  │
│  • 返回 { content, tokens }                    │
└────────────────────────────────────────────┘
                                    │
                                    ▼
响应: { success, message, sources, tokens }
```
