## 一、系统概述

### 1.1 项目简介

师生管理系统是一个基于飞书多维表格（Bitable）构建的在线师生管理平台，提供老师管理、学员管理、班级管理、课时管理、缴费管理、考勤管理等功能模块。

### 1.2 技术栈

| 类别 | 技术 |
|------|------|
| **前端框架** | Next.js 14 (App Router) |
| **后端框架** | Next.js API Routes |
| **数据库** | 飞书多维表格（Bitable） |
| **HTTP客户端** | Axios |
| **开发语言** | TypeScript |
| **测试框架** | Jest + 自定义测试框架 |

### 1.3 项目结构

```
web/
├── app/
│   └── api/
│       └── student-management/      # 师生管理API
│           ├── attendance/          # 考勤管理
│           ├── classes/            # 班级管理
│           ├── courses/            # 课程管理
│           ├── payments/           # 缴费管理
│           ├── student-course-hours/ # 课时管理
│           ├── students/           # 学员管理
│           ├── teachers/            # 老师管理
│           └── utils/               # 工具函数
│               ├── dataProcessor.ts # 数据处理核心
│               └── fieldValidation.ts # 字段验证
├── tests/                          # 测试文件
│   ├── automated-test.ts          # 自动化测试
│   └── comprehensive-test.ts      # 综合测试
└── docs/                          # 文档目录
```

### 3.2 数据处理核心

#### 3.2.1 字段类型定义

```typescript
type FieldType = 'text' | 'number' | 'currency' | 'link' | 'date' | 'checkbox' | 'select' | 'multiSelect';
```

| 类型 | 说明 | 飞书类型 |
|------|------|----------|
| text | 文本类型 | 1001/1005 |
| number | 数字类型 | 1/2 |
| currency | 货币类型 | 2 |
| link | 关联类型 | 11 |
| date | 日期类型 | 5 |
| checkbox | 复选框/多选关联 | 18/21 |
| select | 单选类型 | 3 |
| multiSelect | 多选类型 | 4 |

#### 3.2.2 字段转换规则

```typescript
// checkbox 类型处理
case 'checkbox':
  if (Array.isArray(value)) {
    return value.map(v => v.record_id || v);
  }
  return [value];

// date 类型处理
case 'date':
  if (typeof value === 'number') return value;
  if (value instanceof Date) return value.getTime();
  if (typeof value === 'string') return new Date(value).getTime();
  return undefined;

// link 类型处理
case 'link':
  if (Array.isArray(value)) {
    return value.map(v => v.record_id || v);
  }
  return value?.record_id || value;
```

#### 3.2.3 富文本处理

飞书API返回的文本字段格式为数组，需转换为字符串：

```typescript
// 输入: [{ text: "内容", type: "text" }]
// 输出: "内容"
export function extractText(value: any): string {
  if (!value) return '';
  if (typeof value === 'string') return value;
  
  if (Array.isArray(value)) {
    return value
      .map((item) => item.text || item)
      .filter(Boolean)
      .join(', ');
  }
  
  return String(value);
}
```

## 七、注意事项

### 7.1 字段类型转换

1. **checkbox类型**: 必须传递字符串数组 `["recxxx"]`
2. **date类型**: 传递时间戳或日期字符串
3. **currency类型**: 传递数字或字符串数字
4. **link类型**: 传递record_id或record_id数组

### 7.2 错误处理

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 0 | 成功 | 正常处理 |
| -1 | 通用错误 | 检查msg字段 |
| 99991663 | Token过期 | 刷新token |
| 99991664 | 权限不足 | 检查app权限 |

### 7.3 性能优化

1. **缓存策略**: Access Token缓存25分钟
2. **批量操作**: 尽量使用批量API
3. **分页查询**: 大数据量时使用分页
